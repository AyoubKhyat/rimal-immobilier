# Rimal Immobilier Maroc

Site web premium pour Rimal Immobilier Maroc — agence immobilière de confiance au Maroc.

## Stack technique

- Next.js 15 (App Router)
- GSAP (animations)
- Lenis (smooth scroll)
- React Icons

## Démarrage

```bash
npm install
npm run dev
```

## Production

```bash
npm run build
npm start
```
